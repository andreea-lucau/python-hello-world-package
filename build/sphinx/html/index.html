Welcome to Hello: a Python Template’s documentation!¶

hello¶

This is a sample Python template.

hello.greeting¶

Say a simple hello.

hello.greeting.get_greeting(name)¶

Get a standard hello for the given name.

The name is first validated - it has to be characters-only and non-empty. The hello is: Hello, <name>!

Args:
name: string, the person to say hello to
Returns:
the simple hello
hello.greeting.is_valid_name(name)¶

Check if the given name is valid

A name if consider valid if it is non-empty, contains only characters and the first letter is in uppercase.

Args:
name: string, the name to check for
Returns:
True if the name is valid, false otherwise

hello.random_greeting¶

Say a random hello.

hello.random_greeting.get_random_greeting(name)¶

Get a random hello for the given name.

The name is first validated - it has to be characters-only and non-empty. If the name is valid, read a data file with possible greetings, select one randomly and personalize it with the name.

Args:
name: string, the person to say hello to
Returns:
string, the hello
